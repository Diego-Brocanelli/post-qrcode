# QR Code - PHP

Estrutura do projeto utilizado para implementação dos recursos no post sobre QR Code com PHP.

### Requisitos

    - PHP >= 7.1
    - Ext-db

### Instalação 

```php
composer install
```

### Execução 

```php
php -S localhost:8080 -t public public/index.php
```
